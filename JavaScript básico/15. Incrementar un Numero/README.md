# JavaScript Basico

## Incrementa un número con JavaScript
Se puede incrementar o aumentar un número con el operador " ++ ".

```javascript
i++;
```
Es equivalente a 
```javascript
i = i + 1;
```
Toda la línea se convierte en i++;, eliminando la necesidad del signo de igualdad.
